# GenAI Prompt Engineering Assignment

## Overview

This repository contains my internship assignment on **GenAI Prompt Engineering using LangChain Prompt Templates**. The project demonstrates how to build dynamic and reusable prompt systems instead of using hardcoded prompts.

## Objective

The main objective of this assignment is to understand and implement **PromptTemplate** and **ChatPromptTemplate** in LangChain for creating scalable and reusable Generative AI prompt pipelines.

## Tasks Completed

* Task 1: Replace hardcoded prompts with `PromptTemplate`
* Task 2: Build a multi-input prompt system
* Task 3: Create prompt variations for teaching, interview, and storytelling
* Task 4: Implement `ChatPromptTemplate` with system and human roles
* Task 5: Add input validation for audience and tone
* Task 6: Build a dynamic prompt generator function
* Task 7: Test template reusability with multiple inputs

## Technologies Used

* Python
* LangChain
* Jupyter Notebook / Google Colab

## Project File

* `GenAI_Prompt_Engineering_Assignment.ipynb`

## Expected Pipeline Flow

User Input → Validation → Prompt Template → Dynamic Prompt Generation → Output

## Learning Outcomes

Through this assignment, I learned how to:

* Build reusable prompt systems using LangChain
* Handle multiple dynamic inputs in prompts
* Design prompt variations for different use cases
* Validate user inputs before prompt generation
* Create scalable prompt pipelines for real-world Generative AI applications

## Conclusion

This project helped me shift my thinking from simply writing prompts to designing reusable prompt systems, which is an essential skill in modern Generative AI development.
